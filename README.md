# Landing Page Cliente Todo Dia

Landing page estática em HTML, CSS e JavaScript, pronta para abrir no VS Code.

## Como usar

1. Extraia o ZIP.
2. Abra a pasta `cliente-todo-dia-landing` no VS Code.
3. Abra o arquivo `index.html` no navegador ou use a extensão Live Server.
4. No arquivo `script.js`, troque este link:

```js
const CHECKOUT_URL = "https://seu-checkout-aqui.com";
```

pelo link real do checkout da Kiwify, Hotmart ou plataforma escolhida.

## Onde alterar preço e texto

Os textos principais estão no arquivo `index.html`.
O visual, cores e espaçamentos estão no arquivo `styles.css`.

## Cores da marca

Preto: `#020403`
Branco: `#f5fff7`
Verde neon: `#39ff14`
Verde secundário: `#00d86a`

## Observação

A página já possui vários botões de chamada para ação apontando para o checkout. Enquanto o link real não for configurado, os botões rolam até a seção de oferta.
